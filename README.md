# Café Don Leo - sitio web estático

Propuesta web lista para subir a GitHub y publicar con GitHub Pages.

## Incluye

- `index.html` - página principal
- `tienda.html` - catálogo / tienda
- `nuestro-cafe.html` - historia, origen y atributos del café
- `preparacion.html` - guía de preparación
- `empresas-eventos.html` - pedidos corporativos y eventos
- `contacto.html` - contacto y formulario base
- `styles.css` - estilos globales
- `script.js` - menú móvil y demo del formulario

## Publicación en GitHub Pages

1. Crear un repositorio nuevo en GitHub.
2. Subir todos los archivos de esta carpeta a la raíz del repositorio.
3. Ir a **Settings > Pages**.
4. En **Build and deployment**, seleccionar **Deploy from a branch**.
5. Elegir la rama principal (`main` o `master`) y la carpeta `/root`.
6. Guardar.
7. GitHub publicará el sitio con una URL propia.

## Ajustes recomendados antes de publicar

- Reemplazar teléfono, correo y enlaces reales.
- Sustituir imágenes de referencia por fotos de Café Don Leo.
- Ajustar precios, zonas de envío y medios de pago.
- Conectar el formulario con Formspree, Netlify Forms o backend propio.
- Integrar botón real de WhatsApp.
- Conectar el chatbot a la base FAQ o a un servicio externo.

## Nota

Esta versión es una base visual y funcional. Está pensada para que luego puedas escalarla a:

- carrito real
- checkout con pasarela de pago
- panel administrativo
- chatbot conectado a FAQs
- analítica con Google Analytics y Meta Pixel
